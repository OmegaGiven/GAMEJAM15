# GAMEJAM15 — Campfire Defenders

A 2D co-op tower defense game built in Godot 4.2.

## Concept

You and a friend must defend your campfire from evil spirits closing in from all sides. Gather firewood to keep the fire burning — if it goes out, you lose. Place spirit totems around the campfire to hold back the spirits while you survive long enough to defeat the boss: a reaper-like entity that commands the darkness.

## Core Loop

1. Gather firewood and feed the campfire to keep it lit
2. Place spirit totems to defend against incoming spirits
3. Fight alongside your totems to survive waves of enemies
4. Defeat the boss Reaper to win

## Features

- 2-player local co-op (split-screen, gamepad + keyboard)
- Tower/totem placement with resource management
- Sword combat with animations
- Boss enemy — the Reaper
- Tutorial level

## Built With

- Godot 4.2 (GDScript)
- GL Compatibility renderer
